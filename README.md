# 💅 Jheniffer Silva Lash Designer

> Site institucional moderno, responsivo e otimizado para estúdio de alongamento de cílios. Foco em design sofisticado, experiência fluida e conversão direta via WhatsApp.

 **Live Demo:** [Acesse o site aqui](https://lash-designer-blue.vercel.app/)

---

##  Funcionalidades

- 🪟 **Header Glassmorphism**: Efeito de vidro fosco ao rolar a página
-  **Scroll Suave (Lenis)**: Navegação fluida e leve, sem travamentos
- 📱 **100% Responsivo**: Mobile-first, adaptado para todos os dispositivos
- 💬 **Integração WhatsApp**: Botões flutuantes e CTAs com mensagem pré-definida por serviço
-  **Tabela de Preços Dinâmica**: Seções de Alongamentos e Manutenções atualizáveis
- 🗣️ **Slider de Depoimentos**: Carrossel automático com suporte a touch
- ❓ **FAQ Interativo**: Accordion com perguntas frequentes
- 🖼️ **Galeria Responsiva**: Grid adaptável com hover effects
-  **Performance Otimizada**: CSS/JS minificados, carregamento rápido e SEO básico

---

## 🛠 Tech Stack

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estrutura semântica |
| CSS3 | Estilização, variáveis, glassmorphism e responsividade |
| JavaScript (ES6+) | Interatividade, slider, accordion e controle de scroll |
| Lenis | Biblioteca leve para scroll suave |
| Font Awesome 6 | Ícones vetoriais |
| Google Fonts | Cormorant Garamond + Montserrat |
